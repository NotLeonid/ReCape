ReCape

A simple tool that allows you to re-texture your Minecraft cape without using any mods.

NOTE:
• You must own a real Minecraft cape, because this tool simply replaces the existing cape's texture. It doesn't create a new cape.
• You re-texture the specified cape in general. Example: You changed the look for the Vanilla cape, now anyone's Vanilla cape will appear like the one you chose.
• You re-texture capes only for yourself. No one will see the change, since you're doing that to YOUR client.
• Animated capes are NOT supported. (yet...?)

How it works?

Ever noticed the assets folder in your .minecraft directory? Every texture and sound are stored there. And each has it's own name, and so are the capes. If you head over to assets/skins, you'll find a bunch of folders and files inside. These are all textures of skins and capes of every player you're interacted (ex.: played on a multiplayer server together) with. So all this tools does is it replaces the file in which a certain cape's texture is stored.
